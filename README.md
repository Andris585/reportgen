# ReportGen — Генератор отчётов об очных консультациях

Веб-приложение для автоматической генерации печатных HTML-отчётов из Excel-файлов.  
Работает полностью в браузере — никакой серверной части, никаких загрузок на сторонние серверы.

## 🔗 Использование

1. Откройте сайт по ссылке GitHub Pages (например: `https://andris585.github.io/reportgen/`)
2. Нажмите **«Скачать шаблон»** — получите пустой `.xlsx` с нужной структурой
3. Заполните шаблон данными о консультациях (лист **Отчеты**)
4. Вернитесь на сайт, загрузите заполненный файл (перетащите или через кнопку)
5. Получите готовый отчёт — скачайте HTML или распечатайте в PDF

## 📋 Структура Excel-шаблона

Лист должен называться **Отчеты** и содержать следующие столбцы:

| Столбец | Описание |
|---|---|
| Тип консультации | Очно / Онлайн |
| № консультации | Уникальный номер |
| Дата | Дата проведения |
| Время начало | Время начала |
| Время конец | Время окончания |
| Продолжительность, мин | Длительность в минутах |
| ФИО участника | ФИО консультируемого |
| Комбинат | Название комбината |
| Цех | Название цеха |
| Цех/ Участок | Участок |
| Роль консультируемого | РМ, РГ, ВП и т.д. |
| ФИО консультанта Tactise | ФИО консультанта |
| Инструмент | Вид работы |
| Тема консультации | Тема |
| ИКР по теме консультации | Идеальный конечный результат |
| ИКР продемонстрирован (Выполнено/ Нет/ Частично) | Статус |
| Как продемонстрирован / Почему не продемонстрирован | Описание |
| Рекомендации для самоподготовки | Опционально |

> Количество строк — любое. Пустые поля «Рекомендации» не выводятся в отчёт.

## 🛠 Технологии

- Чистый HTML/CSS/JS — никаких фреймворков
- [SheetJS (xlsx)](https://github.com/SheetJS/sheetjs) — парсинг Excel в браузере
- Полностью автономен — работает без интернета после загрузки страницы
